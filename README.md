# WF-refactoring
SpringBoot ,JPA를 사용한 리팩토링 프로젝트
# WorkFlow 
WorkFlow(전자 결재) 시스템 개발

# 사용 기술
- Back-end : Java, Spring Boot, JPA, Spring MVC,Gradle
- DB : MySQL
- Developer Tools : intelliJ / MySQL Workbench

# 브랜치 관리 :git-flow
```sh
master : 배포용 브랜치
develop : 기능 merge용 브랜치
feature : 기능 개발 브랜치
release : QA 브랜치
hotfix : master 브랜치로 배포 후 수정 브랜치
```
